# 🌐 Domains to Register for reCAPTCHA

## Copy and paste these domains in the Google reCAPTCHA registration form:

registrationportal.local
localhost
127.0.0.1

## Explanation:
- registrationportal.local  → Your IIS production site (main)
- localhost                 → Development testing
- 127.0.0.1                → Local IP testing

## How to Add in Google reCAPTCHA:

1. Go to: https://www.google.com/recaptcha/admin/create
2. In the "Domains" section, add each domain on a separate line:

   ┌─────────────────────────────────────┐
   │ Domains:                            │
   │ registrationportal.local            │
   │ localhost                           │
   │ 127.0.0.1                          │
   └─────────────────────────────────────┘

3. Click "Submit"

## Note:
- These are LOCAL domains for your development and IIS testing
- If you deploy to a PUBLIC domain later (like www.yourcompany.com),
  you can add it by editing the reCAPTCHA site in Google admin console
- All three domains will work with the SAME keys
